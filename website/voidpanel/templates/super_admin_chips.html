How Void Chips Work

  • Signup Bonus — When a new user registers, they receive this many Chips automatically.
  • Exchange Rate — Customers can use Chips to pay invoices. The rate determines how much ₹ value each Chip holds.
  • Example: With a rate of 100 and a bonus of 5000, new users get ₹50 in credit.
  • Chips cannot be used for domain registrations or transfers.
  • Admins can manually adjust a customer's Chips balance from Super Admin → Clients.
