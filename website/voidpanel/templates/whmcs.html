Prerequisites

To begin, you will need an active WHMCS installation and root access to your WHMCS server file structure.

  • Connect via SFTP or SSH to your WHMCS server.
  • Navigate to the modules directory and create a new folder:
[WHMCS ROOT]/modules/servers/voidpanel

Installation Guide

Follow these steps to upload and activate the module:

  1. Download the official VoidPanel WHMCS plugin ZIP file using the button below.
  2. Extract the contents of the ZIP file directly into the voidpanel folder you created in the prerequisites step.
  3. Log into your WHMCS Admin Area.
  4. Navigate to System Settings > Servers and click Add New Server.
  5. Select VoidPanel from the Module dropdown list.
  6. Enter your VoidPanel server IP and root API credentials to establish the connection.
