What's Inside

Everything You Need to Run a Server

A complete suite of server management tools, all included at no cost.

Website Management

Add, configure, and manage unlimited websites on your server with Nginx virtual hosts, PHP version switching, and one-click deployment.

DNS Management

Full BIND DNS control — add A, CNAME, MX, TXT records with an intuitive interface. Manage your entire DNS zone from one place.

Email Hosting

Create and manage email accounts with Postfix + Dovecot. Full Roundcube webmail integration with SSL-secured IMAP/SMTP.

Automatic SSL

Free Let's Encrypt SSL certificates with automatic renewal. One-click SSL activation for all your domains and subdomains.

Database Tools

MySQL/MariaDB management with phpMyAdmin integration, visual database creation, user management, and backup utilities.

CSF Firewall

ConfigServer Security & Firewall integration with brute-force protection, DDoS mitigation, and IP block/whitelist management.

Web Terminal

Browser-based SSH terminal with real-time access to your server. Execute commands, manage files, and monitor logs on the fly.

File Manager

Full-featured web file manager with upload, rename, delete, and permission management. Edit files directly in the browser.

AI Assistant

Built-in AI chatbot to help you with server configuration, troubleshooting, and VoidPanel usage — right inside the panel.

Simple Setup

Up and Running in 3 Steps

Install VoidPanel on your Linux server and take full control in minutes.

1

Install on Your Server

Run a single install script on a fresh Ubuntu or AlmaLinux server. VoidPanel sets up Nginx, Postfix, BIND, MySQL, and all dependencies automatically.

2

Configure Your Panel

Set your hostname, create admin users, and add your first domain. The quick-setup wizard walks you through everything step by step.

3

Manage Everything

Add websites, configure DNS, issue SSL certificates, create email accounts — all from the modern, dark web interface from any browser.
