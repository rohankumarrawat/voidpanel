How to Use This Key
1
Copy the API key from above using the Copy button.
2
Go to your VoidPanel Website → Super Admin → Servers → Add Server Node.
3
Enter this server's URL (e.g. http://YOUR-SERVER-IP:8080) and paste the API key into the API Key field.
4
Click Test Connection — the website will call /api/license/validate/ on this server to confirm the key works, then save the node.
5
Assign the server node to hosting packages in Super Admin → Hosting. When a user pays, the account will be created automatically via /api/provision/create/.
