Engine is set at installation time The web server engine is configured once when VoidPanel is installed via the setup script. Switching engines on a live server requires all domain configurations to be manually converted and services to be restarted — this is a server-level operation, not a panel-level one. If you need to change the engine, do it by re-running the VoidPanel installer on a fresh server.

NGINX Features

  • PHP-FPM FastCGI via Unix sockets
  • Standard config in /etc/nginx/conf.d
  • Native Certbot SSL support
  • MERN/Django reverse proxy
  • Nginx FastCGI caching

OpenLiteSpeed Features

  • Built-in LSAPI — faster PHP than FPM
  • Automatic .htaccess rewrite support
  • HTTP/3 QUIC out-of-the-box
  • LSCache module for object caching
  • Web Admin console on port :7080
