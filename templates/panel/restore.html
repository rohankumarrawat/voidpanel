How Migration Works
Automated provisioning engine
  • Local Backup: VoidPanel unpacks the archive, detects the account structure, and automatically creates the Unix user, domains, and databases.
  • Remote Fetch: Connects via SSH/API to the source panel, downloads the user archive, then restores files, SQL dumps, and email accounts.
  • Conflict Protection: If the domain or username already exists on this server, the migration is aborted to prevent data loss.
  • Background Processing: Tasks run via the Celery async engine — you can safely leave the page after submitting.
